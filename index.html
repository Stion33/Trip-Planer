Backup και χρήση σαν εφαρμογή

Η εφαρμογή αποθηκεύει δεδομένα στον browser. Για επαγγελματική χρήση θέλει backup ή cloud βάση.
Προσοχή: Τα δεδομένα μένουν στη συσκευή/browser σου. Αν καθαρίσεις ιστορικό ή αλλάξεις browser μπορεί να χαθούν. Χρησιμοποίησε backup JSON.
Για εγκατάσταση ως PWA πρέπει να ανοίγει από HTTPS link ή από localhost. Αν το ανοίγεις με διπλό κλικ ως αρχείο, λειτουργεί αλλά δεν εγκαθίσταται.

Πώς το χρησιμοποιείς τώρα

Κάνεις unzip τον φάκελο και ανοίγεις το index.html με διπλό κλικ.
Περνάς/διορθώνεις σκάφη και πορείες από τα αντίστοιχα tabs.
Στον υπολογισμό επιλέγεις σκάφος + πορεία και εξάγεις PDF ή CSV.

Πώς γίνεται κανονική εφαρμογή

Ανέβασμα σε GitHub Pages ή Netlify, ώστε να ανοίγει από κανονικό link.
Μετά από Chrome/Edge: Install app. Σε iPhone: Share → Add to Home Screen.
Για συγχρονισμό σε πολλές συσκευές: σύνδεση με Supabase/Firebase και login.

# SeaTrip

Mobile-first PWA για trip planning, rental ranking, σκάφη, προορισμούς και χάρτες.

## Αρχεία για GitHub Pages

Ανέβασε στο root του repository αυτά τα αρχεία:

- `index.html`
- `manifest.json`
- `sw.js`
- `data.json`
- `icon.svg`
- `icon-192.png`
- `icon-512.png`
- `sea_background.jpg`
- `README.md`

Δεν χρειάζεται `app.js`. Αν υπάρχει παλιό `app.js` από προηγούμενη έκδοση, μπορείς να το σβήσεις για να μη σε μπερδεύει, γιατί ο κώδικας της εφαρμογής είναι μέσα στο `index.html`.

## Δεδομένα

Το `data.json` περιέχει τα κεντρικά/αρχικά δεδομένα της εφαρμογής:

- boats
- routes / destinations
- rentals
- links
- settings

Όταν το app ανοίγει από GitHub Pages, φορτώνει το `data.json`. Οι αλλαγές που κάνεις μέσα από το app αποθηκεύονται τοπικά στο browser με localStorage.

Για να εμφανιστούν ξανά τα δεδομένα του `data.json`, πήγαινε στο Settings/Backup και πάτα **Φόρτωση data.json** ή **Αρχικά δεδομένα**.


## Νεότερες αλλαγές v19

- Αφαιρέθηκε η επιλογή **2D σκάφος / βέλος GPS** από το Live Navigation.
- Προστέθηκε επιλογή **ένωση σημείων** ώστε τα waypoints να ενώνονται μεταξύ τους με γραμμή.
- Το φόντο με τη θάλασσα έγινε λίγο πιο λευκό, σαν πολύ απαλό διάφανο σεντόνι πάνω από την εικόνα.
- Προστέθηκε live υπολογισμός κατανάλωσης καυσίμου στην πλοήγηση: διαλέγεις σκάφος, το app μετρά την απόσταση που διανύεις και εκτιμά τα λίτρα με βάση τη live ταχύτητα και τα στοιχεία κατανάλωσης του σκάφους.
- Ενημερώθηκε το service worker cache σε `seatrip-navigation-v19`, ώστε να μη μένει παλιό cache.

## Νεότερες αλλαγές v18

- Προστέθηκε η φωτογραφία `sea_background.jpg` ως σταθερό φόντο σε όλο το app.
- Ενημερώθηκε το service worker cache σε `seatrip-navigation-v18`, ώστε να μη μένει παλιό cache χωρίς τη νέα εικόνα.

## Νεότερες αλλαγές v17

- Η πυξίδα δείχνει αριθμητικές μοίρες πάνω στην κλίμακα.
- Η κόκκινη απόκλιση ξεκινά από το κέντρο της πυξίδας και μεγαλώνει προς δεξιά ή αριστερά όσο αυξάνεται η παρέκκλιση.
- Αφαιρέθηκαν οι οδηγίες από το κάτω μέρος του χάρτη.
- Προστέθηκε επιλογή **full map** για κινητό/πλοήγηση: ο χάρτης γεμίζει όλη την οθόνη και μένουν μόνο τα όργανα, η πυξίδα, τα σημεία και τα βασικά χειριστήρια πάνω από τον χάρτη.

## Σημείωση cache

Η έκδοση αυτή χρησιμοποιεί `v=17` στα app icons και service worker cache `seatrip-navigation-v17`, ώστε να μη μένει κολλημένη σε παλιό cache.

## Νέα σελίδα Live Navigation

Η έκδοση v16 προσθέτει νέα καρτέλα με εικονίδιο GPS/βέλος. Περιλαμβάνει:

- live στίγμα μέσω Geolocation API
- χάρτη OSM με overlay OpenSeaMap seamarks
- ταχύτητα σε knots, συντεταγμένες, heading, bearing και απόσταση σε nm
- πολλά σημεία πορείας με tap/click πάνω στον χάρτη
- μετακίνηση σημείων με κράτημα και σύρσιμο του marker
- εναλλαγή ενεργού σημείου από select ή με κουμπιά προηγούμενο/επόμενο
- zoom/ξεζουμ με +/- , mouse wheel, διπλό tap και pinch
- ελεύθερη περιήγηση στον χάρτη με σύρσιμο
- λειτουργίες χάρτη: Βορράς πάνω, κεντραρισμένο/heading-up, και βελάκι χαμηλά στην οθόνη
- 2D ένδειξη σκάφους και ευθεία heading line
- κινούμενη πυξίδα τύπου marine/Garmin με live μοίρες heading, ένδειξη στόχου και κόκκινη απόκλιση αριστερά/δεξιά

Σημείωση: το live GPS λειτουργεί μόνο σε HTTPS ή localhost και χρειάζεται άδεια τοποθεσίας από τον browser/κινητό.

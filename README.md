# Captain Stion

Το **Captain Stion** είναι μια mobile-first ναυτική εφαρμογή για προσωπική χρήση σε μικρά σκάφη, σχεδιασμένη για γρήγορο υπολογισμό διαδρομών, καυσίμων, κόστους ταξιδιού, σύγκριση rental σκαφών και live πλοήγηση με GPS.

Η εφαρμογή βοηθά τον χρήστη να οργανώνει θαλάσσιες διαδρομές, να αποθηκεύει σκάφη και προορισμούς, να βλέπει εκτιμήσεις κατανάλωσης και χρόνου, καθώς και να χρησιμοποιεί live χάρτη με σημεία πορείας, ταχύτητα, heading, bearing, απόσταση, ETA και βασικές πληροφορίες καιρού.

## Βασικές δυνατότητες

* Υπολογισμός απόστασης, χρόνου, καυσίμου και κόστους ταξιδιού.
* Καταχώρηση προσωπικών σκαφών με ταχύτητα, κατανάλωση και χωρητικότητα καυσίμου.
* Αποθήκευση προορισμών και διαδρομών σε ναυτικά μίλια.
* Σύγκριση rental σκαφών με βάση κόστος, ταχύτητα, αυτονομία και κατανάλωση.
* Live GPS navigation με χάρτη, waypoints, πυξίδα, bearing, απόσταση και ETA.
* Εκτίμηση live κατανάλωσης καυσίμου κατά την πλοήγηση.
* Βασικές πληροφορίες ανάγκης, σήματα, φώτα, αποστάσεις ασφαλείας και εξοπλισμό.
* Backup και import/export δεδομένων σε JSON.
* Offline αποθήκευση των βασικών αρχείων της εφαρμογής.

## Σημαντική σημείωση ασφάλειας

Το Captain Stion είναι βοηθητικό εργαλείο προγραμματισμού και προσωπικής οργάνωσης. Δεν αντικαθιστά επίσημους ναυτικούς χάρτες, όργανα πλοήγησης, VHF, σωστικό εξοπλισμό, τις οδηγίες του Λιμενικού ή την κρίση του κυβερνήτη.

Πριν από κάθε πλου, ο χρήστης πρέπει να ελέγχει τον καιρό, την κατάσταση της θάλασσας, τα καύσιμα, τον εξοπλισμό ασφαλείας, τους τοπικούς περιορισμούς και τις ισχύουσες ναυτιλιακές οδηγίες.

Οι υπολογισμοί καυσίμου, χρόνου και κόστους είναι ενδεικτικοί και εξαρτώνται από τις πραγματικές συνθήκες, το φορτίο, τον καιρό, την κατάσταση της θάλασσας, τη μηχανή και τον τρόπο οδήγησης.

## Offline χρήση

Η εφαρμογή μπορεί να αποθηκεύσει τα βασικά αρχεία της για χρήση χωρίς σύνδεση. Οι online χάρτες, τα marine overlays και ο καιρός μπορεί να χρειάζονται internet, εκτός αν έχουν ήδη φορτωθεί προσωρινά στην cache της συσκευής.

Για σίγουρη ναυτική offline χρήση, συνιστάται πάντα ξεχωριστή αξιόπιστη λύση offline marine charts.

## Για ποιους είναι

Το Captain Stion απευθύνεται σε χρήστες μικρών σκαφών που θέλουν ένα απλό και γρήγορο εργαλείο για:

* να σχεδιάζουν εξορμήσεις,
* να υπολογίζουν κατανάλωση και κόστος,
* να συγκρίνουν επιλογές ενοικίασης,
* να κρατούν οργανωμένα τα βασικά στοιχεία πλου,
* να έχουν live εικόνα πορείας όταν βρίσκονται στη θάλασσα.

Η χρήση γίνεται με ευθύνη του χρήστη.
